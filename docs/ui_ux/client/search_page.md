## 1. Overview

The search page displays **mixed results** from both Articles and Products, with real-time suggestions, filtering, and sorting capabilities. Powered by MeiliSearch for fast, typo-tolerant search.

## 4. Components

### 4.1 Search Bar
```typescript
interface SearchBarProps {
  query: string;
  onQueryChange: (query: string) => void;
  onSearch: () => void;

  type: 'article' | 'product' | 'category';

  slug: string;
```

- Debounce: 300ms
- Min chars: 2
- Max suggestions: 6
- Auto-focus on page load

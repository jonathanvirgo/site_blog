## 1. Overview

The Article Detail page is the **most important page** for engagement. Users read high-quality content and discover more related articles. The goal is to maximize reading experience while guiding users to explore more content.

### Content Flow
```
Read Article â†’ Discover Related â†’ Explore Category â†’ Personalized Feed â†’ Endless Discovery
```

## 2. User Stories

- As a **reader**, I want to read the full article content without distractions
- As an **engaged user**, I want to discover more articles in the same category
- As a **returning visitor**, I want personalized content recommendations
- As a **mobile user**, I want smooth infinite scroll experience
- As an **SEO crawler**, I want proper structured data for rich snippets

## 4. Components

### 4.1 Article Header
- **Purpose**: Display article metadata
- **Data**: title, category, author, date, views, tags
- **Elements**:
  - Breadcrumb navigation
  - Category badge (clickable)
  - H1 Title (max 3 lines)
  - Author avatar + name (link to author page)
  - Publish date + view count

### 4.2 Sticky Sidebar Products
- **Purpose**: Show related products that follow scroll
- **Data**: Products related to article tags/category
- **Behavior**:
  - Sticky position, `top: 80px`
  - Stops before footer
  - Max 3 products visible
- **Item Design**:
  - Product image (1:1 ratio)
  - Product name (2 lines max)
  - Price
  - "Xem nhanh" button
- **States**: Loading (skeleton), Empty (hidden)
- **Interactions**: Click product â†’ Quick View modal

## 5. Data Requirements

### API Endpoints
| Endpoint | Purpose |
|----------|---------|
| `GET /articles/:slug` | Article content |
| `GET /articles/:slug/same-category` | Same category articles |
| `GET /articles/featured` | Featured/popular articles |
| `GET /articles/notable` | Editor's picks |
| `GET /articles/personalized` | Personalized recommendations |
| `GET /articles/discover?cursor=X` | Infinite scroll feed |

### Data Structure
```typescript
interface ArticleDetailData {
  article: {

    title: string;
    content: string; // HTML

  notableArticles: Article[];

interface DiscoverFeed {
```

## 9. SEO

### Meta Tags
```html
<title>{{article.metaTitle}} | YourSite</title>
<meta name="description" content="{{article.metaDescription}}">
<link rel="canonical" href="https://yoursite.com/article/{{slug}}">
```

### Structured Data (JSON-LD)
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "{{title}}",
  "image": "{{featuredImage}}",
  "datePublished": "{{publishedAt}}",
  "author": {
    "@type": "Person",
    "name": "{{author.name}}"
  }
}
```

### Open Graph
```html
<meta property="og:type" content="article">
<meta property="og:title" content="{{title}}">
<meta property="og:image" content="{{featuredImage}}">
<meta property="article:published_time" content="{{publishedAt}}">
<meta property="article:author" content="{{author.name}}">
```

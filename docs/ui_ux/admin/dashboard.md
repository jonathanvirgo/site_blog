# UI/UX: Admin Dashboard

> **Standard**: This document follows [_wireframe_standard.md](./_wireframe_standard.md)

---

## 1. Overview

The Admin Dashboard provides a comprehensive interface for managing content, products, orders, and site settings. It's designed for efficiency with a focus on common tasks.

### User Roles
| Role | Access |
|------|--------|
| Admin | Full access |
| Editor | Articles, Categories only |

---

## 2. User Stories

- As an **admin**, I want to see key metrics at a glance
- As an **editor**, I want to quickly create and publish articles
- As an **admin**, I want to manage orders and update their status
- As an **admin**, I want to see which articles drive the most sales

### 13.3 Product Editor Components

```typescript
interface ProductEditorProps {
  product?: Product;
  onSave: (data: ProductFormData) => Promise<void>;
  isLoading?: boolean;
}

interface ProductFormData {
  name: string;
  slug: string;
  shortDescription?: string;
  description?: string;
  images: string[];
  categoryId: string;
  tagIds: string[];
  status: 'draft' | 'active' | 'inactive';
  isFeatured: boolean;
  hasVariants: boolean;
  // Single product fields
  sku?: string;
  price?: number;
  salePrice?: number;
  stockQuantity?: number;
  // Variant fields
  attributes?: ProductAttribute[];
  variants?: ProductVariant[];
  // SEO
  metaTitle?: string;
  metaDescription?: string;
}

interface ProductAttribute {
  name: string;
  values: string[];
}

interface ProductVariant {
  sku: string;
  attributeValues: Record<string, string>;
  price: number;
  salePrice?: number;
  stockQuantity: number;
  isActive: boolean;
}
```

### 13.4 API Endpoints

| Endpoint | Method | Purpose |
|----------|--------|---------|
| `GET /admin/products` | GET | List products |
| `GET /admin/products/:id` | GET | Get product detail |
| `POST /admin/products` | POST | Create product |
| `PUT /admin/products/:id` | PUT | Update product |
| `DELETE /admin/products/:id` | DELETE | Delete product |
| `POST /admin/products/:id/variants` | POST | Add variant |
| `PUT /admin/products/:id/variants/:vid` | PUT | Update variant |
| `DELETE /admin/products/:id/variants/:vid` | DELETE | Delete variant |
| `POST /admin/products/:id/variants/bulk` | POST | Bulk update variants |

### 13.5 Validation Rules

| Field | Rule |
|-------|------|
| name | Required, 3-500 chars |
| slug | Required, unique, URL-safe |
| categoryId | Required |
| SKU | Required (per variant), unique |
| price | Required, > 0 |
| salePrice | Optional, < price |
| stockQuantity | Required, >= 0 |
| Attributes | Max 2 per product |
